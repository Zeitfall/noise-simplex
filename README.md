# info

implementation of simplex noise.

# installation

`npm i noise-simplex --save`

```
import { simplex3D, simplex4D } from 'noise-simplex';

simplex3D(
	*float number*,
	*float number*,	
	*float number*			
);

simplex4D(
	*float number*,
	*float number*,	
	*float number*,
	*float number*			
);
```
